Размещение элементов в Grid

Урок 0002 · ~12 минут · Опирается на урок 0001: основы

В прошлом уроке вы научились создавать колонки. Сегодня — главный приз: расставлять элементы по сетке точно туда, куда нужно. К концу урока вы соберёте классический макет «шапка-сайдбар-контент-подвал».

Знание №1: сетка — это линии, а не ячейки

Grid нумерует не колонки, а линии между ними. У сетки из трёх колонок — четыре вертикальные линии (MDN: линии сетки):

B
C

Элемент размещается командой «от линии — до линии»:

.item {
  grid-column: 1 / 3;   /* от линии 1 до линии 3 = первые ДВЕ колонки */
  grid-row: 2 / 3;      /* вторая строка */
}

Полезные сокращения (CSS-Tricks: grid-column):

  • grid-column: span 2 — «займи 2 колонки отсюда»
  • grid-column: 1 / -1 — «от первой линии до последней» = на всю ширину. -1 считает с конца.

Знание №2: именованные области

Для целых макетов линии неудобны. Grid умеет лучше: вы рисуете макет текстом прямо в CSS через grid-template-areas (CSS-Tricks: template areas):

.page {
  display: grid;
  grid-template-columns: 150px 1fr;
  grid-template-areas:
    "header  header"
    "sidebar content"
    "footer  footer";
}
.page > header  { grid-area: header; }
.page > aside   { grid-area: sidebar; }
.page > main    { grid-area: content; }
.page > footer  { grid-area: footer; }

Правила простые: каждая строка в кавычках — строка сетки, каждое слово — колонка. Повторите имя — область растянется на несколько ячеек. Имя каждой области должно образовывать прямоугольник.
