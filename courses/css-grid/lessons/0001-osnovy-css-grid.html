Основы CSS Grid

Урок 0001 · ~10 минут · Миссия: уверенно собирать двумерные раскладки

Сегодня одна-единственная вещь: как любой контейнер превратить в сетку тремя строками CSS. Это фундамент, на котором стоят все макеты — от «шапка-сайдбар-контент» до карточных галерей.

Знание: три строки, которые делают сетку

.container {
  display: grid;                  /* 1. включаем сетку */
  grid-template-columns: 1fr 1fr 1fr;  /* 2. описываем колонки */
  gap: 8px;                       /* 3. зазор между ячейками */
}

Что здесь происходит:

  • display: grid — контейнер становится grid-контейнером, а все его прямые дети автоматически — grid-элементами (MDN: базовые концепции).
  • grid-template-columns — задаёт количество и ширину колонок. Строки создаются сами, по мере заполнения.
  • 1fr — единица доли (fraction): «возьми одну долю свободного места». 1fr 1fr 1fr = три равные колонки (CSS-Tricks: fr unit).

Главное отличие от flexbox: Grid — двумерный (строки и колонки задаются контейнером), flexbox — одномерный (один ряд, который может переноситься). Если вы рисуете макет как таблицу — это Grid.
