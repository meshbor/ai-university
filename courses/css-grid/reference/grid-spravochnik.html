CSS Grid — справочник

Сжатая выжимка уроков 0001 и 0002 · обновляется по мере обучения

Минимальная сетка

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;  /* колонки */
  gap: 8px;                            /* зазор */
}

Все прямые дети контейнера автоматически становятся grid-элементами. Строки создаются сами по мере заполнения.

Размеры колонок (треки)

ЗаписьЗначение
1frодна доля свободного места; 2fr 1fr — первая вдвое шире
150pxфиксированная ширина
autoпо контенту; растягивается, только если в сетке нет fr-треков. Для «контент занимает остальное» предпочитайте 1fr
repeat(6, 1fr)шесть равных колонок

Линии и размещение элементов

Grid нумерует линии между колонками, не сами колонки. У трёх колонок — четыре линии. Отрицательные числа считают с конца (-1 — последняя линия).

ЗаписьЗначение
grid-column: 1 / 3от линии 1 до линии 3 = первые две колонки
grid-column: span 2занять две колонки с текущего места
grid-column: 1 / -1на всю ширину сетки
grid-row: 2 / 4то же самое для строк

Макет областями (template areas)

.page {
  display: grid;
  grid-template-columns: 150px 1fr;
  grid-template-areas:
    "header  header"
    "sidebar content"
    "footer  footer";
}
.page > header { grid-area: header; }
.page > aside  { grid-area: sidebar; }
.page > main   { grid-area: content; }
.page > footer { grid-area: footer; }
  • Каждая строка в кавычках — строка сетки; каждое слово — колонка.
  • Повторённое имя растягивает область. Область обязана быть прямоугольной.
  • Точка . вместо имени — пустая ячейка.

Grid или flexbox?

Grid — двумерный: контейнер управляет и строками, и колонками. Flexbox — одномерный: один ряд с переносами. Рисуете макет как таблицу — берите Grid; раскладываете цепочку элементов в ряд — flexbox.
