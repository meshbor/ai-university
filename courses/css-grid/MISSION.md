# Миссия: CSS Grid

## Зачем

Научиться уверенно собирать двумерные раскладки страниц (шапка/сайдбар/контент, карточные сетки) на CSS Grid, не прибегая к copy-paste и хакам с flexbox. Цель — самостоятельно верстать макет любой сложности с нуля.

## Как выглядит успех

- Могу сверстать классический макет «шапка + сайдбар + контент + подвал» на Grid за 5 минут без подсказок
- Могу сделать адаптивную карточную сетку одной строкой (`repeat(auto-fit, minmax(...))`)
- Понимаю, когда брать Grid, а когда flexbox, и могу объяснить почему

## Ограничения

- Уроки короткие: 10–15 минут каждый
- Язык обучения — русский

## Вне рамок

- Subgrid и контейнерные запросы (container queries) — позже, после основ
- CSS-фреймворки (Tailwind, Bootstrap) — учим чистый CSS
